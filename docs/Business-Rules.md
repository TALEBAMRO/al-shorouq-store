## Retail

- Customers can see retail prices only.
- Customers can place orders directly.
- Customers can manage their addresses.
- Customers can view order history.

---

## Wholesale

- Wholesale customers never see prices.
- They submit quantity requests only.
- Admin reviews every order.
- Admin can edit quantities.
- Admin can remove unavailable products.
- Admin sets prices.
- Invoice is generated after approval.

## Security

- Wholesale prices are hidden.
- Admin only can generate invoices.
- Customers cannot modify approved orders.
