# Al-Shorouq Store v2 Roadmap

## 🎯 Vision

Build a production-ready grocery store management system for Al-Shorouq Store.

The system consists of:

* Retail Store
* Wholesale System
* Admin Dashboard
* Backend API
* Database
* Invoice System
* Reports
* Backup System
